## Project setup

To set up the mbox project, follow these steps:

```
npm install
```

## Development Commands

- **Compiles and hot-reloads for development:**
```
npm run serve
```

- **Compiles and minifies for production:**
```
npm run build
```

- **Lints and fixes files:**
```
npm run lint
```
